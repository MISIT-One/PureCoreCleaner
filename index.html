我們的服務
  1. 出勤車馬費（Call-out Fee）：出勤基本費 NT$ 200，每距離 1 公里加 NT$ 5，未滿 1 公里以 1 公里計算。如 12.3 公里，以 13 公里計算，出勤車馬費為 NT$ 265。
  2. 出勤計算基準起點（Call-out Start Point）：林口社區運動公園（244 新北市林口區文化一路一段 155 號），以我方 Google Map 設定導航，使用汽車，預設路徑為主。
  3. 目前僅收取現金（Cash only），工程師均個人兼職，服務沒有發票。
  4. 與您見面起至整個作業開始結束開機處所會全程進行錄影以利雙方沒有爭議；若您有任何疑慮或不適，請慎重考慮是否要使用我們提供的服務。
  5. 服務的機器主要為 Server / PC。若您是 Apple 的機器，我們非常強烈建議您送原廠處理！
  6. 我們不會去拆解任何風扇、電源、散熱元件來做清潔動作。礙於這些品項有很多刁鑽角度，難以拆解、保固問題，但我們會盡可能將明顯灰塵清除。
